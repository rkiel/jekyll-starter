# Installation

Create a working directory

```unix
mkdir -p ~/GitHub/$USER
cd ~/GitHub/$USER
```

Clone the repository and give it a new name
```unix
git clone git@github.com:rkiel/jekyll-starter.git MY_NEW_BLOG
cd MY_NEW_BLOG
```

Make the repository your own

```unix
rm -rf .git
git init
git add .
git commit -m "Initial commit"
```
